# get_account_count

The get_account_count command outputs the current number of registered accounts.

> Request
```
{
  "jsonrpc": "2.0",
  "method": "get_account_count",
  "params": [""],
  "id": 1
}
```
>Response

```
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": 2116
}
```

> return value

- **result**: Number of accounts
