# witness_node_stop

The witness_node_stop command stops the current ligo_node and ligo_client.

> Request
```
{
  "jsonrpc": "2.0",
  "method": "witness_node_stop",
  "params": [""],
  "id": 1
}
```
>Response

```
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": null
}
```

> Return parameters

- **result**: returns null
